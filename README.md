# Odin Recipes

## Introduction
This project is a simple recipe website built from scratch. It serves as my first major project in **The Odin Project's** Foundations path, allowing me to build a functional multi-page website using raw HTML. 

## Skills Demonstrated
Once this project is fully completed, I will have successfully demonstrated the following skills:

* **Semantic HTML:** Correctly structuring a website using proper headings, paragraphs, and lists.
* **Navigation & Links:** Working with relative file paths to seamlessly connect a homepage (`index.html`) to sub-pages inside a directory (`recipes/`).
* **Media Handling:** Displaying images properly on a web page.
* **Git & Version Control:** Using a standard development workflow by tracking changes, making commits, and pushing updates to a remote repository on GitHub.
